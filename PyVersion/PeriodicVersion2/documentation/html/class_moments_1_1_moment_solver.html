The main integrator that solves the moment and adjoint equations.

Based on a third order Runge-Kutta method This functions goes through each element and integrates the moment equations through a given quadrupole field profile

Parameters
[in]FReference to the function being solved, in this case __ode_moments()
[in]hThe integration step size, see self.h and self.hReversed @paran[in] y0 The initial conditions for the integration.
[in]Thelattice used for the integration
[in]verbosePrint out verbose info while running the integration
[out]toutThe independent variable of the integration
[out]youtThe dependent variable of the integration as a function of tout
[out]kvalThe quadrupole strength as a function of tout
[out]OvalThe O matrix values as a function of tout
[out]NvalThe N matrix values as a function of tout

Main function that solves the moment equations for all 11 moments.

This should not be called directly. Only the integrator ode3() should be calling this as it is integrating through the equations

Parameters
zThe z step to use [m]
YThe value of the 11 moments at the particular z value
dBThe quadrupole gradient [T/m] at the particular z value
rotThe quadrupole rotation angle in radians
Main function that solves Tom's moment equations

Y(1) - Q+
Y(2) - Q-
Y(3) - Qx
Y(4) - P+
Y(5) - P-
Y(6) - Px
Y(7) - E+
Y(8) - E-
Y(9) - Ex
Y(10) - L
Y(11) - phi (solenoid related variable)

Main function that solves the adjoint moment equations for all 10 moments.

This should not be called directly. Only the integrator ode3() should be calling this as it is integrating through the equations

Parameters
zThe z step to use [m]
YtThe value of the 22 moments at the particular z value. This contains the original 11 moments + the 11 adjoint moments for a total of a 22 variable to solve for
dBThe quadrupole gradient [T/m] at the particular z value
rotThe quadrupole rotation angle in radians
